## プロダクト概要

- **プロダクト名**: IRIAM
- **対象ユーザー**: US英語圏のVTuber志望者・VTuberファン
- **提供価値（配信者/ライバー）**: スマートフォン1台でVTuber配信を始められる。PCや複雑なセットアップ不要
- **提供価値（視聴者/リスナー）**: 好きなVTuberとリアルタイムでつながり、ギフティングで応援できる
- **プラットフォーム**: スマートフォンアプリ（iOS / Android）

## ストリーマー種別

配信機能は全ユーザーに公開されておらず、審査を通過した人のみが配信できる。フェーズによってラベルが変わっている。

### フェーズ1（完了）: Founding Streamer
- Atomic Network Phaseで採用した選ばれた配信者
- 審査基準が高く、少数精鋭

### フェーズ2（現在）: Early Access Streamer（EAS）
Founding Streamer終了後に導入した新ラベル。以下の3種類がある。

| 種別 | 略称 | 概要 | ノルマ | 重要度 |
|------|------|------|--------|--------|
| External EAS | EX-EAS | 既にTwitch/YouTubeなどで活動中のVTuberがIRIAMに参加 | なし（任意） | 低（デビュー数目標に含まない） |
| Native EAS | Native EAS | デビューサポートプログラム経由でIRIAMがイラストを無償提供 | 月30時間の配信ノルマ | **最重要** |
| 事務所ストリーマー | - | IRIAMと契約している事務所からイラスト提供を受けてデビュー | Native EASに準ずる | 重要 |

**Native EASが最重要な理由**: 月30時間の配信ノルマによりエンゲージメントが高い。事務所ストリーマーも同様。

### IP権利譲渡ルール

IRIAMがイラストを提供したストリーマー（Native Founding Streamer、Native EAS）に共通のルール。

- **条件**: 累積配信時間が **300時間** に到達した時点で、キャラクターのIP権利がストリーマー本人に譲渡される
- **累積計上の上限**: 1日あたり最大 **2時間** まで計上（例: 1日3時間配信しても2時間分のみカウント）
- **対象**: Native Founding Streamer、Native EAS（＝IRIAMからモデルを提供された全ストリーマー）

### デビューパイプライン（Native EAS）

```
採用（合格通知）
  ↓ 約2〜3ヶ月（イラスト制作期間）※一部は途中で脱落
デビュー
```

- **採用数**が2〜3ヶ月後のデビュー数の先行指標
- 採用→デビューの間に脱落者が出るため、採用数 > デビュー数となる
- **現状**: 採用数 100名達成でデビュー数 約80名を目指す

| 時期 | Native EAS採用数 | 備考 |
|------|----------------|------|
| 2026年1月 | 70名 | 自然応募が50名強を占めた |
| 2026年2月 | 75名 | 目標（100名）まであと一息。2〜3ヶ月後のデビュー数に反映される見込み |

### Native EAS 採用チャネル別状況（2026年2月実績）

**ボトルネック**: 応募数が足りないことが主因

| チャネル | 2月採用数 | 特徴・課題 |
|----------|-----------|------------|
| **自然応募**（LP経由） | 約44名（推定） | 書類通過率は低め。リスナーがIRIAMを使って応募するケースが多いと推測。デジマスケール→リスナー増→自然応募増という連動構造がある |
| **Influencers Club**（スカウトサービス） | 約25名 | メールスカウト。2025年11月開始、徐々に送付数を増やし2月に25名達成。書類通過率は高い。ただし面接ドタキャン・無断欠席が多く、応募→採用の転換率は約50%。月3万件送付・採用転換率約0.1% |
| **Modash**（セルフスカウティングツール） | 約6名 | 自社でのアウトリーチ |
| **アニメコンベンション出展** | 6〜9名 | 毎月全米のアニメコンに出展。Discord に50〜60名集めることには成功。コンバージョン目標は10名/回だが現状は6〜9名/回 |

### Influencers Club 改善施策（予定）

- **現状の課題**: Influencers Club 独自ドメインからのメール送付 → 受信者に「スキャムでは？」と疑われるケースが多い
- **対策**: IRIAMのメールアドレスをInfluencers Clubに提供し、IRIAMドメインから送付する形に変更
- **期待効果**: コンバージョン率の改善 → 月間採用数 +10〜15名程度が目標
- **現在の数値感**: 月3万件送付・採用転換率 約0.1%。わずかな率の改善でも絶対数への影響は大きい

### 採用チャネルの構造的なつながり

```
デジタルマーケティング出稿額スケール
  → リスナー（DAU）増加
    → 自然応募増加（リスナーがストリーマーに応募）
      → Native EAS採用数増加
        → デビュー数増加 → DLU増加
```

デジタルマーケティングは**リスナー獲得**と**ストリーマー応募増**の両方に効いている二重の効果がある。

### デジタルマーケティング出稿額スケール計画（直近）

| 月 | 出稿額（月額） |
|----|--------------|
| 1〜2月 | 500万円 |
| 3月 | 600万円 |
| 4月 | 750万円 |
| 1年後 | 2,000〜2,500万円（長期計画） |

## 配信開始の前提条件（重要）

IRIAMで配信するにはVTuberキャラクターが必要:
- **Live2D モデル**（動くキャラクター）
- **PNG（PNGtuber）**（静止画キャラクター）

このハードルを下げるため、**デビューサポートプログラム**を運営。IRIAMがキャラクターイラストを無償提供することで、ゼロからでもVTuber配信を始められるようにしている。**現在の主力施策**。

### グロース構造（因果関係）

```
デビューサポートプログラム
  → ストリーマーデビュー数の増加
    → DLU（Daily Liver User）の増加
      → DAUの増加
```

直近数ヶ月でこのサイクルが回り始め、DLU・DAUともに上昇傾向にある。

## ARPU向上戦略

**目標**: 現状ARPUの2〜2.5倍を達成する

IRIAM-JPでの歴史的実績を強く参考にしている。JP → US への適用は、USユーザー特性に合わせてアレンジしながら進める方針。

### 参考: IRIAM-JPでの実績

- **2019年**: アプリ内イベント数の大幅増強 + ライバー数の拡大 + 多様なギフト追加（毎月追加でギフティングカルチャーを醸成）+ リアルイベント（ニコニコ超会議出展、大型スクリーン配信企画等） → **1年間でARPUが約2倍**に向上
- **2020年**: コミュニティランクシステム導入 → ARPUのじわじわとした向上

### 柱1: イベント充実 ＋ ギフト定期追加

- アプリ内イベントを定期的・継続的に充実させる
- 毎月新しいギフトを追加し、ギフティングを楽しむカルチャーを強化する
- JP版で実績のあるアプローチをUSにも適用

### 柱2: コミュニティランクシステム（導入予定）

JP版（2020年導入）を参考に、US版向けにアレンジして導入予定。

**ランクの仕組み**:
- ストリーマーのコミュニティの盛り上がり（ギフト以外の参加度合い等）をもとにランクが上下する

**US版での活用方針**（JP版との差異）:
- JP版: ランク上昇 → 「時間ダイヤ」（配信時間に応じた報酬）が発生
- US版: 時間ダイヤは導入しない。代わりに**アプリ内イベントの参加条件**としてランクを活用（例: 魅力的なイベントへの参加はAランク以上が条件）
- ランクへの上昇動機 → 配信頻度・エンゲージメント向上 → ARPUの漸進的な向上を狙う

## 新規ユーザー獲得（NUU）の認知経路

### ユーザーサーベイ結果（サインアップ後2日目に送付）

複数回答あり。

| 認知経路 | 割合 |
|----------|------|
| ストリーマー経由（口コミ・SNS等） | 50〜60% |
| デジタル広告 | 約40% |
| 切り抜き動画 | 約20% |

### NUU数値上の獲得経路

- **デジタルマーケティング経由**が大多数を占める
- オーガニック経由は少数

### チャネル別の定着率・絶対数

| チャネル | 定着率 | 絶対数への寄与 |
|----------|--------|----------------|
| デジタル広告 | 低め（US市場全般の傾向） | ボリュームが大きいため、定着ユーザー全体に占める絶対数は多い |
| オーガニック | （相対的に高いと推定） | 母数が少ない |

- 定着率は低いが**獲得ボリュームの大きさで絶対数をカバー**しているのがデジタル広告の実態
- オーガニック vs 広告の定着ユーザー数の正確な内訳は要確認

### デジタルマーケティングの現在の運用方針

- **管理指標**: CPI（Cost Per Install）を主軸に予算を決定・執行
- **ROIについて**: 現フェーズでROIを強く意識すると施策が萎縮するため、意図的に優先度を下げている
- **フェーズ認識**: 現在は先行投資フェーズ。ユニットエコノミクスの最適化より規模拡大を優先

### デジタルマーケティング スケール計画

事業シミュレーションに基づく計画。ARPUの向上計画と合わせて事業構造が成立する見通し。

| 時期 | 出稿額（月額） | CPI目標 |
|------|----------------|---------|
| 現在 | 約500万円 | 600円台 |
| 1年後 | 2,000〜2,500万円 | 約1,000円 |
| 2年後 | 約5,000万円 | 1,300〜1,500円程度（要確認） |

- スケールに伴いCPIは上昇する前提で計画を立てている
- デジタルマーケティングの拡大単体では成立せず、**ARPU向上との両輪**で事業構造が成立する設計
- 事業シミュレーション上の見通しであり、前提条件の変化に応じて見直しが必要

### 解釈の仮説（未確定）

デジタルマーケティングが「最後の後押し」として機能しているマルチタッチのパターンと推測。
ストリーマーの発信・切り抜き・口コミ等で複合的に認知が積み上がり、最終的にデジタル広告で獲得に至るケースが多いのではないか。

### データ解釈上の留意点 ⚠️

- サーベイ回答者はオーガニック獲得ユーザーの比率が高いと予想される
- そのため**サーベイの認知経路割合 ≠ NUU全体の認知経路割合**
- デジタルマーケティング経由のNUUの認知経路は、このサーベイでは十分に捉えられていない可能性がある

## DSP Bootcamp

デビューサポートプログラム（DSP）参加者の中から希望者を募り、1ヶ月間の集中トレーニングを行うプログラム。

### 概要

- **目的**: DSP参加ストリーマーのパフォーマンスの底上げ
- **期間**: 1ラウンド = 約4週間
- **対象**: DSP参加者の中から希望者を募集
- **実施実績**: Round 1〜3まで完了（R3は2026-02-26終了）

### プログラム構成

- 参加者ごとにDiscordチャンネルを用意
- 週次レポートの提出（最大7回/ラウンド）
- トラッキング指標: IRIAMフォロワー、Xフォロワー、配信回数（30分以上）、配信時間、初回訪問者数、2回目訪問者数、Tier1ファンバッジ獲得数
- ラウンド終了時にパーソナライズされたProgress Summaryを送付

### 実績サマリー

| 指標 | Round 2 (n=19) | Round 3 (n=17) |
|------|---------------|---------------|
| 期間 | 2025/11/28〜12/25 | 2026/1/30〜2/26 |
| IRIAMフォロワー増（平均） | +80.8 | +62.8 |
| Xフォロワー増（平均） | +66.6 | +46.5 |
| 配信回数 30min+（平均） | 27.2回 | 28.4回 |
| 配信時間（平均） | 42.6h | 39.3h |
| 初回訪問者（平均） | 150.3 | 137.6 |
| 2回目訪問者（平均） | 51.0 | 42.4 |
| Tier1ファン獲得（平均） | 11.7 | 7.2 |
