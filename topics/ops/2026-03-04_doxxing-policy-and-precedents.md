# ドキシング（個人情報無断公開）対応方針と事例記録

**日付**: 2026-03-04
**領域**: オペレーション / コミュニティ安全
**ステータス**: `進行中`

---

## 概要

IRIAMストリーマーによる他ストリーマーの個人情報無断公開（ドキシング）について、過去事例と現在進行中の案件をまとめ、対応方針の基準を整理する。

---

## 事例1: Icarus → Meimei（2025年9月）

### 事実関係

| 項目 | 内容 |
|------|------|
| 行為者 | Icarus Hakamori（Founding Streamer） |
| 被害者 | Meimei（Founding Streamer） |
| 行為 | Meimeiの顔写真を交際相手（未成年）にDMで共有 |
| 共有先 | 1名（交際相手） |
| 共有手段 | Discord DM |
| 行為者の主張 | 交際相手からの圧力で共有した。悪意はなかった |
| 写真の入手経路 | Meimeiが別の友人（Miuna）に共有→Miunaが拡散→Icarusが入手 |

### 対応プロセス

1. **9/13**: Meimeiから報告受領（ドキシング + 未成年との関係の2件）
2. **9/15**: Meimeiとオンラインミーティング実施
3. **9/21-22**: 追加証拠（写真共有のスクリーンショット）受領
4. **9/23**: Meimeiに対応方針を説明、被害者の意向確認（今進めるか待つか）
5. **9/24 AM**: Icarusへヒアリング通知（回答期限: 9/28）
6. **9/24 昼**: Icarus即日回答。事実を認めつつ「交際相手の圧力」を主張
7. **9/24 PM**: **処分決定 — 厳重警告 + 2週間配信停止**

### 処分内容（ドキシング単体）

- 厳重警告
- 2週間の配信停止 + IRIAMキャラクター使用禁止（〜10/8）
- 違反した場合はFS地位の永久剥奪と警告

### 被害者の反応

- **処分に不満を表明**
- 「ドキシングした人と同じプラットフォームで配信し続けることへの精神的負担」
- IRIAM離脱を検討するレベルの不満
- 最終的に継続を選択したが、不満は残った

### その後の展開

- **9/29**: 未成年との性的関係の証拠が別途確定
- **処分**: FS地位剥奪 + キャラクターライセンス終了（実質永久BAN）
- ※この永久BANはドキシングではなく未成年との関係が理由

### 振り返り・教訓

- ドキシング単体の2週間停止処分は、被害者にとって不十分と受け止められた
- 「交際相手に友人の顔写真を見せた」という行為は、動機として理解可能な範囲ではあった
- 結果的にIcarusは別件（未成年関係）で永久BANとなったため、ドキシング処分の妥当性は問い直されずに終わった

## 事例2: Lioran → Nightshade（2026年3月）【進行中】

### 事実関係

| 項目 | 内容 |
|------|------|
| 行為者 | Lioran Arden（Native EAS） |
| 被害者 | Nightshade（Founding Streamer） |
| 行為 | Nightshadeの個人情報をGoogle Driveにまとめ、第三者に共有 |
| 共有された情報 | 顔写真（複数、幼少期含む）、摂食障害の闘病、愛犬の死など極めてプライベートな内容 |
| 共有先 | 少なくとも1名の視聴者（POKER鬼）。Nightshadeは「viewers」と複数を示唆 |
| 共有手段 | Google Driveリンク（Discord DMで共有） |
| 行為者の主張 | 「記録目的」「ドキシングの意図なし」「修正した」（実際は未修正） |
| Google Driveの現状 | アクセス制限はかかっている（確認済み） |

### 証拠の状況

- **Lioran本人がDMで共有を自認**（「あのスクリーンショットとリンクはあなたにだけ共有した」「ドライブにアップロードした」）
- Nightshadeから証拠スクリーンショット提供済み
- 事実自体に争いはない

### 対応プロセス

1. **3/3**: Nightshadeから報告受領
2. **3/3**: プロフィールリンクと証拠の提供を依頼
3. **3/4**: スクリーンショット・Google Driveリンク受領、証拠確認
4. **3/4**: 社内で対応方針を協議 → 永久BAN方針で合意
5. **3/4**: Nightshadeへ中間報告（「調査中、完了したら連絡する」）
6. **3/4**: Lioranへヒアリング通知送付（回答期限: 3/5）
7. **3/6予定**: 回答確認→処分実行（EASステータス剥奪・Character License Agreement停止・アカウント永久凍結）

### 前例（Icarus）との差異 — なぜより重い処分が妥当か

| 観点 | Icarus（2週間停止） | Lioran（永久BAN予定） |
|------|---------------------|----------------------|
| **共有された情報** | 顔写真1枚 | 顔写真複数（幼少期含む）＋ 病歴（摂食障害）＋ 私的な苦悩 |
| **共有手段** | DMで個別送付 | Google Driveにフォルダとして体系的に整理・保存 |
| **共有先** | 交際相手1名 | 視聴者（少なくとも1名、複数の可能性） |
| **行為の動機** | 交際相手からの圧力（理解可能な範囲） | 「記録目的」で網羅的にアーカイブ→第三者に提供（理解しがたい） |
| **事後対応** | — | 「修正した」と虚偽の主張、実際は未修正 |

**判断の根拠**:
1. **情報のセンシティブ度**: 幼少期の写真・病歴は個人情報の中でも最も保護されるべきカテゴリ
2. **行為の性質**: 「うっかり見せた」ではなく、意図的に他人の個人情報をアーカイブして第三者に提供する行為
3. **前例の反省**: Icarus案件で被害者（Meimei）が処分に明確な不満を示した教訓

## プラットフォーム外行為への対応について

- ドキシングはDiscord / Google Driveなどプラットフォーム外で発生するケースがほとんど
- IRIAMストリーマー同士の関係性の中で発生している以上、プラットフォームとして対応する立場にある
- 過去にプラットフォーム外行為で処分した前例あり（Icarus案件）

---

## 対応基準の整理（今後の参考）

| 行為の重大度 | 例 | 想定される処分 |
|-------------|------|---------------|
| **軽度** | 公開済み情報の転載など | 警告 |
| **中度** | 非公開の顔写真を少数に共有（Icarus相当） | 厳重警告 + 活動停止（2週間〜） |
| **重度** | センシティブ情報の体系的な収集・公開（Lioran相当） | EASステータス剥奪・永久BAN |

※ 上記は目安であり、個別の状況（意図・範囲・被害者への影響等）に応じて判断する
