**Colored rough sketch > Line art > Rendering**

- **Phase 1: Color Rough**
Here we can accommodate up to two sets of changes. These can be changes to the clothes themselves, adding smaller additional accessories, or touching up the hair! The biggest changes happen here, so please let us know if there are things you'd like to add once we get the draft to you!
**We can accept up to two sets of revisions at this stage**, so if the you'd like extra revisions after seeing an edited draft, we're all ears!

- **Phase 2: Line Art**
The illustrator will clean up the outlines from the rough draft, allowing for more clarity for smaller details. Here we can adjust the size of accessories and the finer details of clothes! Here, **we can only accommodate one set of adjustments**, so please bear that in mind 🙏

- **Phase 3: Final Render**
The illustrator will color in the outline and render the piece. Here we can adjust the colors, brightness, and saturation, but we cannot make adjustments to the design itself. Similar to the Line Art phase, **we can only accommodate one set of adjustments.**

In order to ensure a proper schedule and workflow for our illustrators, we'd like to request a response to each draft within **3 business days** to avoid delays in production 🙏
